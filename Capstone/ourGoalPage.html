Our Goal

In this section, we will describe our goal for this project which is to be able to help small businesses design a website and be able to have the top website protection for the sensitive information they hold.

The purpose of this project is to be able to give small businesses a website that is able to fully protect the vulunerabilties that are involved in a website. As well as, allow users to conect with each other through the use of a google form on the Sign Up page which allows contact information to be shared amongst both Small Businesses and Helpers. Helpers should navigate to the Individuals page if they would like to sign up to help businesses. Examples of these vulnerabilities would include SQL injections, as well as, weak passwords for employees or customers, URL rediction to bad websites, and code downloads without the correct intergity checks.
